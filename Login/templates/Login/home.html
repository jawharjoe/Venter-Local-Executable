            CATEGORY PREDICTION MODEL

            The “Category Prediction Model” tab in the navigation bar helps the user to return a set of top three
            predicted complaint categories for an input CSV file, wherein the output generated is in a tabular format.

            1.  User can upload CSV file to the portal with the correct headers
            2.  The size limit for input CSV file is 12 MB
            3.  Initial processing of the file takes about a minute, hence the user is expected to be patient while the
                portal retrieves the predicted results
            4.  Some of the languages used in the output predicted categories (e.g. Hindi, Marathi) won’t be displayed when
                downloaded and viewed on MS Word in Windows. To resolve the same, the user can download Marathi script
                online to view those categories
